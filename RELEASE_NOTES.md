# v0.5.17

## New

- **11 new themes** — Neon Sunset, Polar, Reactor, Amber, Macchiato, Shibuya, Solarized Dark, Evergreen, Cobalt, Minimal Dark, and Transilvania
- **Theme hover preview** — Hover over any theme in Settings or the status bar picker to instantly preview it before committing
- **Themes grouped by Dark and Light** — Easier to browse with labeled sections and a separator
- **Activity bar hover labels** — Hover any sidebar icon to see its name expand as a pill, with a subtle shift effect
- **Markdown files open in the main area** — Clicking a `.md` file in the file explorer now shows a rich rendered preview with edit mode, formatting toolbar, and Mermaid diagram support
- **Markdown formatting toolbar** — Bold, italic, headings, lists, links, tables, and more with toolbar buttons and keyboard shortcuts (Cmd/Ctrl+B, I, K)
- **Default plugins** — New installations automatically include essential plugins (currently Markdown Preview)
- **Frosted Dark is now the default theme** for new installations

## Fixes

- **Voice input on macOS** — Added microphone permission so voice features in AI coding assistants can request mic access
- **Plugin shell commands** — Fixed an issue where plugin shell commands always appeared to fail
